• Question : What is component ?
    • Answer : Component is basic buiding block in angular, using that we can create specific functionality For example : in normal website we have basic three parts available like header, body and footer ok. so we can create 3 different component for each three part.
    • We can create header component, body component and footer component and all three component have its own specific functionality.
    • Component contain mainly 4 file,
      • .css file : using this file we can set style for that pericular component html
      • .html file : in this file we can create our html design
      • spec.ts file : using this we can test our component
      • component.ts file : in this file we can create and use property,logic and query connection or pass data
    • For create component we just need to run below commad

      ng generate component "Component name" OR ng g c "Component name"
